# Kshitij Soni — Portfolio

Personal portfolio website for Kshitij Soni, Vehicle Dynamics Simulation Technical Lead.

## Files

- `index.html` — the website (open in any browser, no build step)
- `KVS_profile_photo.jpg` — profile photo used in the About section
- `Kshitij-Soni-Vehicle-Dynamics-Technical-Lead.pdf` — resume, linked from the Resume download button
- `images/` — project gallery images (referenced by the slideshow)

## Publishing with GitHub Pages

1. Create a public repository (e.g. `yourusername.github.io` for a clean URL).
2. Upload all files **and the `images/` folder** to the repository.
3. Go to **Settings → Pages**, set Source to **Deploy from a branch**, choose **main** / **/(root)**, and Save.
4. Your site goes live at `https://yourusername.github.io` within a minute or two.

## Updating

Edit `index.html` (or replace images/resume), commit the change, and the live site updates automatically.

## Adding more project images

In `index.html`, find the `PROJECT_IMAGES` object inside the `<script>` block.
Each project lists its image file paths in order. Add new files to the `images/`
folder and list their paths to extend a gallery.
